BreakMate Installer Return Codes

BreakMate uses NSIS (Nullsoft Scriptable Install System) for its Windows installer. Below are the EXE return code values for standard installation scenarios.

Scenario Return Code Description
Installation successful 0 The installation completed successfully.
Installation cancelled by user 1602 The user cancelled the installation before completion.
Application already exists 1638 Another version of the application is already installed. The installer will offer to update it.
Installation already in progress 1618 Another installation is already in progress. The user must complete or cancel it first.
Disk space is full 112 Not enough disk space to complete the installation.
Reboot required 3010 Installation succeeded but a system restart is required to complete the process.
Network failure 1231 A network error occurred during the installation.
Package rejected during installation 1260 The package was blocked by a security policy on the device (e.g., AppLocker, SmartScreen).
Miscellaneous install failure 1603 A general fatal error occurred during installation.

Silent Installation

BreakMate supports silent installation with the /S switch:

BreakMate_1.4.8_x64-setup.exe /S
