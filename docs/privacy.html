Privacy Policy

Last updated: February 16, 2026

In short: BreakMate does not collect, store, or transmit any personal data. The application works entirely offline and requires no accounts, no internet connection, and no permissions beyond basic desktop access.

1. Overview

BreakMate is a desktop application for Windows that provides health reminders (eye rest, hydration, posture, movement, and stretching). This privacy policy explains how BreakMate handles your data.

2. Data Collection

BreakMate does not collect any personal data. Specifically:

  • No personal information is collected (name, email, phone, etc.)
  • No usage analytics or telemetry are collected
  • No browsing data or activity is tracked
  • No location data is accessed
  • No device identifiers are collected
  • No crash reports are sent

3. Network Activity

BreakMate operates 100% offline. The application makes zero network requests. It does not connect to any servers, APIs, or cloud services. No internet connection is required to use BreakMate.

4. Local Data Storage

BreakMate stores a small amount of data locally on your device to remember your preferences:

  • Settings: Reminder intervals, language preference, and display options
  • Statistics: Daily reminder counts and water intake tracking
  • Registry: Installation path and autostart preference

All data is stored in the following locations:

  • Application files: %LOCALAPPDATA%\BreakMate\
  • Registry keys: HKCU\Software\BreakMate
  • Autostart entry: HKCU\Software\Microsoft\Windows\CurrentVersion\Run\BreakMate

5. Third-Party Services

BreakMate does not integrate with any third-party services, SDKs, or analytics platforms. The application does not include any advertising frameworks.

6. Cookies

BreakMate does not use cookies. The application does not run in a browser context and does not store or access browser cookies.

7. Accounts & Authentication

BreakMate does not require any account creation, login, or authentication. There is no user registration process.

8. Children's Privacy

BreakMate does not collect any personal data from anyone, including children. The application is safe for users of all ages.

9. Data Deletion

Since all data is stored locally on your device, you have complete control over it:

  • Uninstall the application to remove all BreakMate data from your device
  • Silent uninstall: "%LOCALAPPDATA%\BreakMate\uninstall.exe" /S
  • The uninstaller removes all application files and registry entries

10. Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top. Since BreakMate does not collect any data, significant changes to this policy are unlikely.

11. Contact

If you have any questions about this privacy policy or BreakMate's data practices, you can open an issue on our GitHub page.

Our commitment: BreakMate was built with privacy as a core principle. We believe your health data and personal information belong to you alone. That's why we designed BreakMate to work entirely offline, with zero data collection.
